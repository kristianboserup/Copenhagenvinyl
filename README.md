# Copenhagenvinyl

Minimalist grayscale landing page for Copenhagenvinyl — curated vinyl, bicycles, clothing and furniture from Copenhagen.

- `index.html` — front page with category grid
- `vinyl/` — 9 genre pages, 18 artist pages, 54 record detail pages and 27 label pages
- `img/` — place hero/category photos here (`hero.jpg`, `vinyl.jpg`, …) and artist photos in `img/artists/` (`rock-1.jpg`, …). Pages fall back to hosted images until the local files exist.

Static site — no build step. Open `index.html` or serve the folder (works with GitHub Pages).
